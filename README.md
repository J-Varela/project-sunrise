# Project Sunrise

Project Sunrise is a lunar mapping simulation project.

The mission concept is to use orbiters to map permanently shadowed lunar craters before sending robots into them. These regions are scientifically valuable but dangerous because they are dark, extremely cold, and difficult to navigate.

## Learning Goals

- Learn GMAT for orbital mission simulation
- Learn Python data parsing for space mission outputs
- Build a Next.js dashboard to visualize mission data
- Later add ROS 2 for robotics communication
- Later add Blender or Isaac Sim for 3D robotic simulation
